# TETRIS
TETRIS with JavaScript  
  
https://ohzzi.github.io/TETRIS/
  
------------------------------------------------------------------  
  
How To Use

↑ : Rotate  
↓ : Move down  
← : Move left  
→ : Move right  
SPACE : Hard drop  
ESC : Pause
  
------------------------------------------------------------------  
  
The high score is saved in a browser cookies for 10 days.  
If you want to reset your high score, clear your browser's cookies.
